📋 Kod Statusları

Gözləmədə

Kod admin tərəfindən yoxlanılmağı gözləyir. Bu zaman kod axtarış səhifəsində görünməz.

  • Admin təsdiqi gözləyir
  • Kod satışa hazır deyil
  • 24 saat ərzində yoxlanılır

Təsdiqləndi

Kod admin tərəfindən təsdiqlənib və aktiv satışdadır. İstifadəçilər kodu ala bilər.

  • Kod aktiv satışdadır
  • Baxış və satış statistikası görünür
  • Gəlir hesablanır

Bloklandı

Kod admin tərəfindən bloklanıb. Səbəb: qaydalara uyğunsuzluq və ya şikayət.

  • Kod satışdan çıxarılıb
  • Yeni satış üçün aktiv deyil
  • Blok səbəbi: qaydalar pozuntusu

Bitmiş

Kodun etibarlılıq müddəti bitib. Avtomatik olaraq satışdan çıxarılıb.

  • Etibarlılıq tarixi keçib
  • Kod avtomatik deaktiv olub
  • Yenilənməsi mümkün deyil
